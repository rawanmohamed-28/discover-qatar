# Discover Qatar Tourism Website

<h2>Project Overview</h2>
The <strong>Discover Qatar</strong> website is an informational tourism website aimed for tourists who want to explore Qatar and learn more about the country, from its climate to cultures. The website hilights popular places to visit and helps them plan their trip, from transportation to hotels, and booking a flight. The webite has four main pages: tho home page, about Qatar page which hilights its climate and culture, Places To Visit which is divided into: restaurants, souqs, shopping malls, and sights (landmarks), and the last page is called 'Plan Your Trip' which helps the user plan their trip such as trasnportation or renting a car, to booking a hotel stay, and an airline ticket, using external websites.
The website user friendly.
